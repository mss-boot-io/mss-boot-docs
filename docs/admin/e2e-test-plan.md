## 测试工具

### 后端测试
```bash
# 运行所有后端测试
go test ./... -v -cover

# 运行特定模块测试
go test ./service -v -cover
go test ./apis -v -cover
```

### 前端测试
```bash
# 运行前端单元测试
pnpm test

# 运行E2E测试
pnpm e2e
```

### API测试
```bash
# 使用 curl 进行API测试
curl -X POST http://localhost:8080/api/v1/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"123456"}'
```
