## 概述

本文档定义 `mss-boot-admin` 的全量测试用例，覆盖：

- 认证授权
- 用户管理
- 角色权限
- 组织架构
- 菜单配置
- 日志管理
- 系统监控
- 存储管理
- 任务管理
- 告警通知

> **测试时间**：2026-04-03
> **测试版本**：产品打磨三轮完成后
> **测试类型**：集成测试 + 功能测试

## 测试环境

### 后端
- 服务地址：`http://localhost:8080`
- 数据库：SQLite（默认）/ MySQL / PostgreSQL
- Redis：可选（WebSocket 集群需要）

### 前端
- 访问地址：`http://localhost:8000`
- 浏览器：Chrome / Firefox / Safari

### 测试账号
- 管理员：`admin` / `admin123`
- 普通用户：测试时创建

## 模块 1：认证与授权

### 1.1 登录功能

#### TC-AUTH-001：账号密码登录 - 成功
**前置条件**：用户已注册
**测试步骤**：
1. 访问登录页面 `/user/login`
2. 输入正确用户名 `admin`
3. 输入正确密码 `admin123`
4. 点击登录按钮
**预期结果**：
- 登录成功
- 跳转到首页 `/`
- 顶部显示用户信息

**前置条件**：用户已注册
**测试步骤**：

- 登录失败
